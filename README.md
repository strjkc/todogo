# Todogo

Just a small practice cli Todo app for https://roadmap.sh/projects/task-tracker. 0 to nil fanciness.
### Usage:
- _add "Buy groceries"_
- _update 1 "Buy groceries and cook dinner"_
- _delete 1_
- _mark-in-progress 1_
- _mark-done 1_
- _list_
- _list done_
- _list todo_
- _list in-progress_
